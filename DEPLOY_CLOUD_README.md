## 🚀 快速开始（推荐 Vercel）

### 1. 创建 Supabase 数据库
```
访问：https://supabase.com
→ New Project
→ 等待初始化（2分钟）
→ 复制数据库连接字符串
```

### 2. 部署到 Vercel
```
访问：https://vercel.com
→ Add New Project
→ 导入你的 GitHub 仓库
→ 添加环境变量：
   DATABASE_URL = [你的 Supabase 连接]
   WALLET_ENCRYPTION_KEY = [生成的密钥]
→ 点击 Deploy
```

### 3. 生成加密密钥
```bash
openssl rand -base64 32
```

### 4. 完成！
等待 2-3 分钟，访问你的网站。
